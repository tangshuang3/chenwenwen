致未来的你：

近来可还好？

你是否在想念三月笑春风的桃花，那六月泛着涟漪的湖面，九月飘落的枯叶和那十二月结冰的枝桠。

我不太记得了，是因为那普通又美好的一面在我匆匆路过时悄然错过。

但我仍记得你，记得你穿着白净的裙子在风中飘摇，随着风经过我的身边，你没注意到我，我只是路上的行人一个。

再见你是在哪里呢，不是在熟悉的道路上，也不是在承载记忆的课堂中，是梦里，最美好也不最不真实的地方。

六月的夏风是炎热的风，是离别的风，是悲伤的风，也是高兴的风。

六月的到来标志着一段经历的逝去，预兆着一个未来的开始，人们互相助兴，互相告别，互相慰勉，相约在下一个六月一定相见。

大家背起行囊，有人远航，有人不惧艰难再来一场，有人迫不得已抗下生活的重担。

那你呢，那时你在干什么呢？

可能当你看见时已是过去许多年后，此时你又在干什么呢？你身上青梅味的香气是否还在，你是否仍然笑容依旧

你我皆在感慨过去的美好，回忆曾踏过的草地，驰骋过的球场。好像我们的生活不一样，但经历却相同。

萧瑟的秋风孕育不了娇艳春花，刺骨的寒冬里长出孤傲的梅。人间尽是孤寂事，何不当个不怨仙？风有风的忧伤，树有树的忧愁。你我未曾见，何必犯伤愁。

崔护曾写下一首诗

去年今日此门中，

人面桃花相印红。

人面不知何处去，

桃花依旧笑春风。

于我而言，你正是那人面，也是那桃花。

最让我印象深刻的是2021年8月14日，七夕的那一天。你的生日一直记得很清楚，每年七夕的后一天。那时你因为疫情原因无法去长沙集训，于是有了更多时间来回复我发的消息。 当你在七夕凌晨告诉我，你会放一天假时，我的内心是忐忑，是不安的。我骨子里的懦弱让我难以撑起勇气约你出去，我会担心你是否会拒绝我。 经过煎熬的思想斗争后，我还是在朋友的怂恿下鼓起了勇气。

而你最初给的回应，“看情况吧。”

人的脆弱大概就是这样，当你以为一切顺利的时候，偏偏突然出现的打击让你猝不及防。不得不说，那时我感觉心脏好像暂停了。仿佛一把利刃刺进了跳动的心脏，它没有流血，而是像玻璃一样破碎，散落一地而消失不见。

幸运的是，平时嘻嘻哈哈的损友会在此刻安慰我，重新燃起我的内心。

经过好友的一番思想输出后，我决定一不做二不休，利用“道德绑架”式的形式。于是一大早，我就跑到电影院去购票，幸运的是，电影院在那天刚好解封，也许冥冥中有天意。当时仅有两部电影，我现在仍然记得，一部是青春校园剧《夏至未至》，一部是犯罪嫌疑剧《怒火 重案》。因为拿不准，我特意还问了下别人哪部电影好，结果遭到别人的批骂“七夕节去电影院不看爱情片，那就是真的奇葩。”

买好票后，我马上把照片发给了你，似在逼迫你。你没有马上回应，不知是在犹豫还是确实未看见消息。在我不断地信息轰炸下，你终于回了消息“再看看吧，下午得去做蛋糕。”

电影开始的时间是晚上七点，我特意去借了部电动车。时间越来越近，越来越近。天公不作美，临近六点钟时，天空开始飘起了雨。我看着下得越来越大的雨，心里竟没起一丝波澜，没有可惜，没有后悔，我好像接受了老天对我的嘲弄。

“好像开始下雨了，你准备去吗？”

我已经不抱希望了，纯属是找个交代，或者是找个理由可以让我把票扔掉。

明明内心没有波澜，但在发完消息出去后，我心里开始难过。就像《龙族》中的路明非一样，被设在局中，成为别人的陪衬品。那一刻我终于理解了路明非，我和他都是衰小孩。

“好啊。”

让我不可思议是你竟然同意了，荒凉的沙漠长出了娇艳的鲜花。

“你现在在哪，我去接你”

“我在……，但好像下雨了。”

已经六点半了，我拿着伞马上跑出去找出租车，并在路上要了司机的号码。终于，在广场的一处找到了你。

你身着一身黑裙，给我的第一印象是，优雅，如黑天鹅一般。笨拙的我匆匆跑去接你，由于太着急，我竟然自己先上了车，让你在外面淋了雨。真是很蠢啊。

终于，我们赶到电影院已是七点多了，由于第一天解封，来看电影的人不多，除了你我就剩一对夫妻了。

影院里黑沉沉，只有画面在不断地闪动。我时常瞄向你，你在认真看电影，我在认真看你。

影片很快就结束了，时间也到了九点。“你着急回家吗？”我问。

你说不着急，我又说“那去喝杯奶茶怎么样，反正现在还在下雨”。你答应了，并说要请我喝，我有点不好意思，可是我知道我囊中已经羞涩，我怕等会付不了款的尴尬于是答应了下来。

我们就这样聊了一个多小时，看着时间到了十点钟，已经不早了，我又特意打了出租车司机的电话。

淋了点雨，司机的车内开着冷风。短衣短裤的我一上车便打起了寒颤，我不敢再开口说话，我怕声音都是哆嗦的。很快，车就到了。把你送下车后，我把伞留给了你。我知道你还要走一段路。噢差点忘记了给你的生日礼物，那是一本《哈利波特》的立体书。你说你未曾看过《哈利波特》，但我希望这个礼物能让你爱上它，因为它真的很有趣。

回到家洗完澡后，我等待着零点的到来，我拿起我的破木吉他，为你谈了一首蹩脚的生日歌。

“生日快乐！”
